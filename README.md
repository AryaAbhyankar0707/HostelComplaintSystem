# HostelComplaintSystem
The Hostel Complaint Management System is a Java-based application that allows users to register, view, and resolve hostel complaints. It uses OOP, collections, and file handling to manage data efficiently, improving transparency, organization, and tracking of student issues.
## Hostel Complaint Management System

##  Problem
Students face issues in hostels but cannot track complaints properly.

##  Solution
A Java-based system to register, view, and resolve complaints.

##  Features
- Register complaint
- View complaints
- Resolve complaints
- Data stored using file handling

##  Technologies Used
- Java
- OOP
- Collections
- File Handling

##  How to Run
javac Main.java
java Main
